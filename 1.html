링크를 만들기 위해서는 'a' 태그를 넣으면 된다. 새창에서 열리게 하고 싶으면 "target" 속성을 넣으면 된다. 단축키 쓰는 법
  1. CTRL+D : 원하는 단어를 잡고, 컨트롤디를 누루면 반복된 글자를 한번에 잡음
  2. ALT+click : 알트 누르고 클릭하면 한번에 여러 곳을 잡아서 수정 가능
  3. ALT + 화살표 : 줄을 잡고 알트화살표로 움직이면 그 줄을 위 아래로 이동시킬 수 있음
  4. ALT + SHIFT + 화살표 : 줄을 잡고 알트쉐프트화살표로 움직이면 그 줄을 복사 할 수 있음
  5. CTRL + / : 주석 처리
  6. ATL + SHIFT + I : 여러 줄을 고르고 누르면 한번에 수정 가능
  7. ATL + SHIFT + DRAG : 움직이는 곳 마다 커서가 생김
